# 起跑控制

## 说明

起跑控制（Launch Control）是一项竞速起步辅助功能。飞手可让四轴飞行器前倾，再松开摇杆，飞行器会在起跑位置保持姿态；随后加油门即可起跑。此功能从 Betaflight 4.0 起提供。

## 安全

四轴飞行器处于起跑控制状态时，表面上可能显得安静、安全，实际并非如此。应将其视为随时可能猛烈冲出的高功率飞行器。绝不要松开控制器，随时准备解除解锁；不要靠近，也不要让其他人靠近。请遵守所有常规安全要求。

## 起跑流程

- 启用起跑控制模式开关；
- 解锁；
- 保持最低油门；
- 将四轴飞行器向前俯仰至所需角度；
- 将摇杆回中；
- 推动油门起跑。

示例视频：https://youtu.be/_S0rn3zcN0Q

## 模式开关配置

必须配置该模式才能启用起跑控制。模式状态会在解锁时锁定，因此飞手可灵活配置：使用专用开关、解锁时按住的瞬时开关、直接绑定解锁开关，甚至始终启用。

## 参数

**`launch_control_mode`**：可设为 `NORMAL`（默认）、`PITCHONLY` 或 `FULL`。

- `NORMAL`：用于以底挂电池在地面上平衡起跑。横滚和俯仰会保持位置。可控制偏航，但 PID 不会尝试保持偏航位置。
- `PITCHONLY`：用于竞速起跑架，或可能用于顶挂电池的离地起跑。完全禁用横滚和偏航；同时让前电机保持怠速，以减少从起跑架跌落的机会。若尝试用电池平衡起跑，切勿使用此模式，因为前电机不会响应，飞行器会向前倾倒。
- `FULL`：与 `NORMAL` 相同，但也保持偏航位置。请谨慎使用，长时间保持可能导致偏航 I 项累积。

**`launch_trigger_allow_reset`**：可设为 `OFF` 或 `ON`（默认）。

决定起跑触发器的重置行为。起跑触发后，功能应被关闭，使飞手即使坠机后重新解锁，也不会再次启用起跑控制。

- `ON`：飞行器上锁时，可将模式开关关闭再重新开启来重置起跑触发器。若模式开关可独立选择启用与关闭，请使用此方式。
- `OFF`：起跑控制无法重置，直到飞控断电重启或重新启动。若无法独立关闭该模式，例如它始终启用或绑定到解锁开关，请使用此方式。

只要尚未触发起跑，飞手可多次解锁 / 上锁，功能仍可用。例如飞行器从起跑架跌落后，可重新放置并再次解锁。

**`launch_trigger_throttle_percent`**：范围为 `0` 至 `90`，默认 `20`。

该参数设置油门死区：低于该值时起跑控制保持活动；油门超过设置百分比时触发起跑。数值越高，起跑越激进，因为油门会“跳”至配置百分比。设为 `0` 时，触发值为 `min_check`。默认值 `20` 能避免意外触发，并提供良好的起跑跃升。使用较高值时务必谨慎，飞行器起跑可能非常激进。

**`launch_control_gain`**：范围为 `0` 至 `200`，默认 `40`。

决定用于位置保持的 I 项增益。若飞行器难以保持位置，可提高该值。高值可能导致 I 项累积：如果位置保持期间电机持续加速，增益很可能过高。应使用能接受的最低值。飞行器抬起至起跑姿态时，轻微移动属正常现象，尤其易受风影响；通常不必为了消除少量移动而提高增益。

## 加速度计集成

启用加速度计后，OSD 会显示当前俯仰角，飞手可据此微调起跑角度以保持一致性。

**`launch_angle_limit`**：范围为 `0` 至 `80`，默认 `0`。

可选设置前倾角限制，帮助获得一致的起跑角度。此项只是防止超过限制后继续前倾；飞行器不会主动或自动转到该角度，飞手仍可将角度调低。设为 `0` 即关闭限制。

## OSD 集成

若已启用 Warnings 元素，起跑控制激活时会显示 `LAUNCH` 指示。还需启用起跑控制警告选项（`osd_warn_launch_control = ON`）。若已启用加速度计，指示后会附加当前俯仰角，例如 `LAUNCH 25`。

**4.2 新增：**当油门进入触发百分比的 10% 范围内时，OSD 警告会开始闪烁，提示即将触发起跑。

可在 OSD 菜单的 PROFILE -> MISC PP -> LAUNCH CONTROL 下调整所有起跑控制参数。

## 注意

出现以下任一情况时，起跑控制不会激活：

- 在 Acro 以外的飞行模式下解锁；
- 已启用 `MOTOR_STOP`，且解锁后电机未旋转（Airmode 关闭）；
- 已启用 3D 功能；
- 使用摇杆解锁。
