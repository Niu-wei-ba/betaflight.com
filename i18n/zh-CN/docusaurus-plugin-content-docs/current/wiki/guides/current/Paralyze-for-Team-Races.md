# 团队赛的 Paralyze 模式

团队接力赛中赛道持续处于热区，坠机四轴飞行器不宜立即回收。若主机仍在场地内通电，要安全起飞备用机，需要：

- 禁止主机解锁，防止备用机起飞时意外解锁。某些发射机可向所有通电接收机发送信号，例如 FrSky ACCST D8/D16。
- 将 VTX 切换至低功率、未使用的“墓地”频道，避免干扰备用机。

Betaflight 3.5 引入 `paralyze` 模式以禁止解锁。该模式只能在四轴未解锁时触发，并且是锁定模式：激活后必须断电再上电才能退出。

触发该模式前，模式开关必须至少处于一次未启用状态，避免主机坠机后忘记复位开关而立即让备用机进入 Paralyze。

Paralyze 仅负责禁止解锁。要同时使 VTX 进入并保持 Pit Mode，可在 CLI 将两种模式关联：

```text
aux 2 39 2 1700 2100 0 45
```

此处 AUX 通道（2）和范围（1700-2100）不重要；命令末尾的 `45` 表示模式 39（VTX Pit Mode）关联到模式 45（Paralyze）。45 激活时，39 也会激活并与 Paralyze 一样保持锁定。

请在 CLI 中配置 Pit Mode 频率，确保其工作正常：

```text
set vtx_pit_mode_freq = 5917
```
