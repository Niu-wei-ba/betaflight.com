# 声呐

声呐传感器可测量高度，供 `BARO` 和 `SONAR` 定高模式使用。

在低空（距地面约小于 3.5 米）时，声呐传感器会替代压力传感器（气压计）。仅当飞行器的倾角（姿态）较小（小于 `22.5°`）时，才会使用声呐传感器。

## 硬件

目前主要支持 HCSR04 传感器。Parallax 28015 单线声呐也可使用：在 Trigger 和 Echo 引脚上各串联一个 `1k` 电阻，然后将两个电阻的另一端短接，并接到声呐模块。

```
          1k
TRIGGER--/\/\--\
                \_______ 28015 SONAR
          1k    /
ECHO-----/\/\--/
```

## 连接

### Naze/Flip32+

| 模式                    | Trigger       | Echo          | 串联 `1k` 电阻  |
| :---------------------- | :------------ | :------------ | :-------------- |
| 并行 PWM/ADC 电流传感器 | PB8 / Motor 5 | PB9 / Motor 6 | 否（耐受 5V）   |
| PPM/串行接收机          | PB0 / RC7     | PB1 / RC8     | 是（3.3V 输入） |

#### 限制

电流计不能与并行 PWM 和声呐同时使用。

### CC3D

| Trigger   | Echo      | 串联 `1k` 电阻  |
| :-------- | :-------- | :-------------- |
| PB5 / RC4 | PB0 / RC5 | 是（3.3V 输入） |

#### 限制

声呐不能与 SoftSerial 或并行 PWM 同时使用。

### SPRacingF3

| Trigger   | Echo      | 串联 `1k` 电阻  |
| :-------- | :-------- | :-------------- |
| PB0 / RC7 | PB1 / RC8 | 是（3.3V 输入） |

#### 限制

声呐不能与 SoftSerial2 或并行 PWM 同时使用。
