# GPS 选项卡

在此配置 GPS。Betaflight 使用 GPS 实现 GPS Rescue，并可在 OSD 显示坐标。

![GPS 选项卡](/img/betaflight_configurator_gps_tab_zh-cn.png)

## GPS 配置

| 选项                         | 说明                                                                                                                                     |
| ---------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| **协议**                     | GPS 协议。**UBLOX**：最常见，多数 GPS 模块使用此协议；**NMEA**：用于不支持 UBLOX 的模块；**MSP**：用于通过 MSP 协议传输 GPS 数据的模块。 |
| **自动设置**                 | 自动配置 GPS，建议保持启用。M10 GPS 模块需要 Betaflight 4.5 或更新版本。                                                                 |
| **使用 Galileo 系统**        | 启用 Galileo 卫星系统；多数用户建议启用。                                                                                                |
| **设置单次返航点**           | 启用后，仅使用接入电池后的首次解锁位置作为返航点；未启用时，每次解锁都会更新返航点。                                                     |
| **地面辅助类型**             | 选择地面辅助类型，应与所在地区匹配。                                                                                                     |
| **磁偏角度**                 | 磁力计磁偏角，仅 GPS 模块带磁力计时有用。                                                                                                |

## GPS 信息

| 选项                             | 说明                                                                                                       |
| -------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| **3D 定位**                     | 是否获得三维定位；GPS Rescue 和 OSD 显示坐标均需要此状态。                                                 |
| **卫星数**                       | 当前用于定位的卫星数量。                                                                                   |
| **高度**                         | GPS 上报的高度。                                                                                           |
| **速度**                         | GPS 上报的速度。                                                                                           |
| **IMU / GPS 航向**               | IMU/GPS 上报的航向。                                                                                       |
| **当前纬度 / 经度**              | 当前经纬度。                                                                                               |
| **距返航点距离**                 | 至返航点的距离。                                                                                           |
| **位置 PDOP**                    | 当前[位置精度衰减因子](<https://en.wikipedia.org/wiki/Dilution_of_precision_(navigation)>)；数值越低越好。 |

## GPS 信号强度

此区域显示各 GPS 卫星的信号状态。

| 选项                | 说明                       |
| ------------------- | -------------------------- |
| **GNSS ID**         | 卫星所属 GNSS 的 ID。      |
| **卫星 ID**         | 卫星 ID。                  |
| **信号强度**        | 卫星信号强度的可视化表示。 |
| **状态**            | 该卫星是否正在用于定位。   |
| **质量**            | 信号质量。                 |
