# 飞行前环境选项卡

“飞行前环境”选项卡提供实时环境数据，帮助您在飞行无人机之前做出明智的继续/停止决策。 **无需连接飞控**即可使用 - 只需打开 Betaflight 地面站并单击侧栏中的“预飞行”即可。

所有数据均来自免费的开源 API。不需要 API 密钥或帐户。

---

## 设置您的位置

在加载任何数据之前，您需要使用以下两种方法之一设置您的航班位置：

**浏览器地理位置** — 单击 **“使用我的位置”** 并允许浏览器访问您的位置。这会使用您设备的 GPS 或网络位置。

**手动输入** — 在输入字段中输入纬度和经度值，然后单击 **应用**。坐标必须采用十进制度（例如，荷兰鹿特丹的 `51.9225` 和 `4.4792`。

设置位置后，所有环境数据都会自动加载。

### 保存的地点

您最多可以保存 **5 个航班地点**以便快速访问：

1. 设置位置（通过地理位置或手动输入）并等待数据加载
2. 单击 **保存** 按钮（书签图标，如果您已经保存了位置，则单击 **+**）
3. 输入位置名称（最多 20 个字符）并确认

您保存的位置显示在下拉列表中。 **选择保存的位置会自动加载它** - 应用坐标并立即获取所有数据。使用垃圾桶图标删除已保存的位置。

保存的位置存储在浏览器的本地存储中，并在会话中保留。

## 启动状态

顶部的醒目横幅显示了总体航班推荐：

| 状态             | 意义                                     |
| ---------------- | ---------------------------------------- |
| **GO**（绿色）   | 一切条件都有利于飞行                     |
| **注意**（黄色） | 有些条件是微不足道的——飞行前查看详细信息 |
| **警告**（橙色） | 重大问题——飞行时要格外小心或推迟         |
| **禁止**（红色） | 飞行条件不安全                           |

该状态是根据风速、能见度、降水量和太阳活动计算得出的。当您单击“**刷新**”时，它会更新。

### 个人支票

横幅下方，详细显示了每项检查及其自己的状态图标：

| 检查       | 来源      | 它评估什么           |
| ---------- | --------- | -------------------- |
| **风**     | 开放气象  | 地面风速和阵风       |
| **可见性** | 开放气象  | 水平可视距离         |
| **降水**   | 开放气象  | 目前有雨、雪或毛毛雨 |
| **太阳能** | NOAA SWPC | Kp 指数/地磁扰动     |

每项检查都会显示一个彩色图标 - 绿色检查（良好）、黄色/橙色感叹号（警告/警告）或红十字（危险） - 因此您可以快速查看哪种特定情况正在推动整体状态。

## 当前天气

显示您所在位置的实时天气状况：

| 字段         | 对飞行的影响                                                                          |
| ------------ | ------------------------------------------------------------------------------------- |
| **风速**     | 地面风。高于 11 m/s 的值很强；对于大多数无人机来说，超过 14 m/s 的速度是危险的        |
| **阵风**     | 峰值风速——突然的阵风可能导致失控                                                      |
| **可见性**   | 你能看到多远。 1 公里以下为差； VLOS（视觉视线）规则通常需要清晰的可见度              |
| **降水**     | 雨、雪或毛毛雨。任何降水都会增加电子和光学的风险。                                    |
| **露点**     | 形成冷凝的温度。当气温接近露点（2-4℃ 以内）时，预计会出现雾气和镜片起雾。             |
| **云层**     | 天空被云层覆盖的百分比                                                                |
| **感觉就像** | 考虑风寒和湿度的表观温度——与飞行员舒适度和设备操作相关                                |
| **湿度**     | 相对湿度 — 高值与温度下降相结合会导致凝结                                             |
| **压力**     | 大气压力（hPa）——快速变化可能表明即将到来的天气锋面                                   |
| **电池风险** | 锂聚合物电池温度评估。低于 10°C：容量降低；低于 0°C：损坏风险。 40°C 以上：有肿胀风险 |
| **雾风险**   | 基于露点扩散、湿度和风速的雾概率。露点附近平静、潮湿的条件表明雾风险较高              |

颜色编码值：**绿色** = 良好，**黄色** = 中等，**橙色** = 警告，**红色** = 危险。

数据来源：[Open-Meteo](https://open-meteo.com/)（免费、开源、无需 API 密钥）。

## 飞行窗口

显示用于计划会话的基于时间的条件：

| 领域             | 描述                                                                                                                    |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------- |
| **日出/日落**    | 大多数国家的合法飞行仅限于白天。这些时间可以帮助你计划                                                                  |
| **合法飞行窗口** | 民用黄昏窗（日出前 30 分钟至日落后 30 分钟）。当您当前在窗口内时为绿色，当您在窗口外时为红色。注意：法规因国家/地区而异 |
| **日光持续时间** | 可用日光总小时数和分钟数                                                                                                |
| **紫外线指数**   | 太阳紫外线强度。高紫外线 (6+) 会使屏幕在户外难以阅读                                                                    |
| **温度范围**     | 每日最低和最高温度——极端寒冷会影响锂聚合物电池性能                                                                      |
| **目前**         | 无论该地点当前是白天还是夜间                                                                                            |

数据来源：[Open-Meteo](https://open-meteo.com/)每日预报参数。

## 海拔高度风预报

显示高于地面 (AGL) 三个高度的风况的 12 小时每小时表：

| 专栏     | 描述                                                    |
| -------- | ------------------------------------------------------- |
| **10m**  | 地面风（您在地面上感受到的风）                          |
| **80m**  | 中空风 — 与大多数 FPV 和商业航班相关                    |
| **120m** | 高空风——对于远程飞行很重要（120m 是许多国家的法定上限） |
| **阵风** | 地面峰值阵风                                            |
| **雨%**  | 该小时的降水概率                                        |

风速的单位是**米每秒 (m/s)**。根据经验：

- 低于 5 m/s：风平浪静，适合飞行- 5-8 m/s：微风，适合大多数无人机
- 8-11 m/s：中等——较小的无人机可能会遇到困难
- 11-14 m/s：强——仅飞行重型/强力飞机
- 超过 14 m/s：对于大多数无人机来说都是危险的

获取 2 天的预测窗口以保证至少 12 小时的未来数据，无论一天中的什么时间。

数据来源：[Open-Meteo](https://open-meteo.com/) 多个压力级别的每小时预测。

## 5 天预测

简洁的每日概览可帮助您提前计划飞行行程。显示未来 5 天的天气状况、温度范围、风力、阵风和降水概率。

| 专栏     | 描述                                 |
| -------- | ------------------------------------ |
| **日**   | 工作日和日期                         |
| **天气** | 带图标的天气状况（晴、阴、雨、雪等） |
| **温度** | 每日温度范围（最低/最高 °C）         |
| **风**   | 当日最大风速（米/秒）                |
| **阵风** | 当日最大阵风速度（米/秒）            |
| **雨%**  | 当日最大降水概率                     |

行根据整体飞行能力进行颜色编码：

- **无亮点**：飞行条件良好
- **浅黄色**：中等 — 风速 8-11 m/s 或降雨几率高于 30%
- **浅橙色**：警告 - 风速 11-14 m/s 或降雨几率高于 50%
- **浅红色**：危险 — 风速高于 14 m/s 或降雨几率高于 80%

数据来源：[Open-Meteo](https://open-meteo.com/)每日预报参数。

## 太阳活动

太阳和地磁活动直接影响 GPS 精度。太阳的活动会干扰 GPS 信号穿过的地球电离层。

### Kp 指数

**行星 K 指数 (Kp)** 测量 0-9 级的地磁扰动：

| Kp 值 | 水平             | 对 GPS 的影响                      |
| ----- | ---------------- | ---------------------------------- |
| 0-2   | 低（绿色）       | 影响最小 — GPS 救机可靠            |
| 3-4   | 3-4 中等（黄色） | 可能会出现轻微退化                 |
| 5     | 升高（橙色）     | GPS 精度可能会明显降低             |
| 6-9   | 风暴（红色）     | GPS 干扰严重 — 不推荐使用 GPS 救机 |

### NOAA 风暴等级

从 0（无）到 5（极端）的三个等级：

- **G（地磁风暴）** — 影响 GPS 精度和指南针可靠性
- **S（太阳辐射）** — 影响卫星电子设备
- **R（无线电中断）** — 影响高频无线电传播

数据来源：[NOAA 空间天气预报中心](https://www.swpc.noaa.gov/)（免费、公共 JSON 源）。

## GNSS/GPS 状态

### 海拔

您所在位置的地面海拔（以米为单位，高于平均海平面 (AMSL)），从 Open-Meteo Elevation API 获取。对于计算绝对高度上限非常有用（例如，450m AMSL = 570m AMSL 中的 120m AGL）。

### 密度高度

密度高度表示螺旋桨感觉空气的“稀薄”程度，根据海拔、压力和温度计算得出。更高的密度高度意味着更少的升力和更高的功耗。包含在启动状态检查中。

| 密度海拔     | 状态 | 影响         |
| ------------ | ---- | ------------ |
| < 1000m      | 好   | 性能正常     |
| 1000–2000 米 | 中等 | 略微降低升力 |
| 2000–3000 米 | 警告 | 性能明显下降 |
| > 3000m      | 危险 | 升力显着降低 |

### 电离层影响

显示当前太阳条件如何影响 GPS 性能的简单语言评估。

### GPS 救机可靠性

指示在给定的当前地磁条件下 Betaflight 的 GPS 救机功能是否可靠。当 Kp > 5 时，GPS Rescue 的位置精度可能会降低，从而导致返航行为不可靠。

### 磁偏角

您所在位置的真北与磁北之间的角度，使用**世界磁模型 (WMM)** 计算得出。如果使用磁力计，这是您应该在 Betaflight 罗盘配置中设置的值。

### 磁倾角

地球磁场的倾角。在磁极附近，高倾斜角可能会导致指南针问题。

### GNSS 规划工具

链接到 [gnssplanning.com](https://www.gnssplanning.com/)，获取您所在位置和时间的详细卫星能见度预测。

磁数据：使用 [geomagnetism](https://github.com/naturalatlas/geomagnetism) 库计算客户端（NOAA WMM 模型，无需调用 API）。

## 空域和禁飞区

外部空域信息服务的直接链接，在支持的情况下预先填写您的位置：

| 链接                             | 覆盖范围                     | 描述                                                             |
| -------------------------------- | ---------------------------- | ---------------------------------------------------------------- |
| **无人机安全地图**               | 全球（超过 155 个国家/地区） | Altitude Angel 提供的免费地图，显示空域限制、地面危险和控制区域  |
| **SkyVector 空域图**             | 全球                         | 显示受控空域、机场和禁区的航空 VFR/IFR 剖面图                    |
| **FAA 航行通告（美国）**         | 美国                         | 美国联邦航空局 (FAA) 官方通知飞行员搜索 — 检查临时飞行限制 (TFR) |
| **欧洲航空安全航行通告（欧盟）** | 欧洲                         | 临时空域限制的欧洲官方航行通告数据库                             |

**重要提示：** 在飞行前务必检查当地航空当局的规定。无人机法规因国家和地区而异。
