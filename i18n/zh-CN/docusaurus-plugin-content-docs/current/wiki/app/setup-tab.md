# 设置选项卡

此页提供基本设置和飞控信息，也是连接飞控后显示的首个选项卡。可在此校准传感器、检查陀螺仪实时预览、查看禁止解锁标志和其他飞控状态。

![设置选项卡](/img/betaflight_configurator_setup_tab_zh-cn.png)

## 基本设置

### 校准加速度计

加速度计用于确定飞控姿态，包括俯仰、横滚和偏航角。按界面提示完成校准。

:::info

加速度计服务于 Angle、Horizon、Acro Trainer 和 GPS Rescue 等自稳功能。若不使用这些功能，可跳过校准，甚至禁用加速度计。

:::

### 校准磁力计

磁力计（罗盘）用于确定飞控在三维空间中的航向，可提高 GPS Rescue 的航向信息准确性。按提示完成校准。

磁力计校准有时间限制，只有 30 秒可完成规定动作。

### 擦除设置

:::danger

此操作会将**全部**设置恢复默认值，得到空白配置。它不是飞行器或飞控的“恢复出厂设置”，也不会仅恢复刚改动的设置。该操作相当于硬重置，可能造成不可预期结果，甚至需要重新刷写才能再次连接。

:::

### 激活 bootloader/DFU

无需按实体 BOOT 键即可手动让飞控进入 DFU（Device Firmware Update）模式。刷写遇到问题时很有用。板卡会保持 DFU 模式，直到重新断电上电或完成刷写。

## 飞控信息

### 陀螺仪实时预览

实时显示陀螺仪数据，用于检查陀螺仪是否正确对齐及是否存在明显噪声。

:::info

可在“配置”选项卡设置陀螺仪朝向。

:::

### 信息

显示飞控的基础数据：

- **禁止解锁标志**：显示阻止飞控解锁的全部状态，用于排查无法解锁。
- **电池电压**：电池电压，需正确配置电压检测。
- **消耗电流**：电池电流，需正确配置电流计。
- **已用容量**：已消耗容量，需正确配置电流计。
- **RSSI**：当前 RSSI（粗略反映信号强度），需正确配置 RSSI。

### GPS

飞控已连接并配置 GPS 时，显示以下数据：

- **3D 定位**：是否获得三维定位；正常 GPS 功能需要定位成功。
- **卫星数**：当前锁定的卫星数；通常 6 颗或以上才有较好的定位。
- **纬度/经度**：飞行器当前经纬度。

### 仪表

以图形方式模拟真实航空仪表。

### 实验性备份与恢复

可备份和恢复地面站界面能修改的配置；无法备份仅能通过 CLI 修改的设置。
