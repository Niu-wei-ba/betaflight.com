# 预设选项卡

预设以简短 CLI 命令片段的形式快速配置飞控，涵盖调参预设和特定产品设置。也可创建并向社区分享自定义预设。

![预设选项卡](/img/betaflight_configurator_presets_tab_zh-cn.png)

## 搜索过滤器

可使用以下条件缩小预设范围：

- **类目**：常见预设类别，包括：
  - **FILTERS**
  - **LEDS**
  - **MODES**
  - **OSD**
  - **OTHER**
  - **RATES**
  - **RC_LINK**
  - **TUNE**
  - **VTX**
- **关键词**：按预设名称和说明搜索关键词。
- **作者**：按作者筛选。
- **固件**：按固件版本筛选。
- **状态**：按状态筛选：
- **官方**：由 Betaflight 团队制作和/或正式支持，通常更高级或针对特定设置。
- **社区**：由社区用户制作，常用于特定产品（如图传表）和飞手速率。
- **实验性**：仍在开发和测试中。

## 预设预览

搜索结果显示预设基本信息；打开后可查看完整说明和选项。

### 结果预览

显示预设名称、说明、作者、固件版本和状态。

![预设预览](/img/betaflight_configurator_preset_1.png)

### 预设详情

除结果预览内容外，还包含：

- **选项**：可配置的选项，用于改变预设行为，例如为调参预设选择电机 KV，或为接收机预设选择链路选项。
- **说明**：预设及其选项的详细说明，包括警告和建议。
- **讨论**：预设 Pull Request 链接，可查看并参与讨论。
- **在线查看**：预设文件链接，可查看预设代码。
- **显示命令行**：查看选中预设将应用的 CLI 命令。
- **关闭**：关闭预设详情。
- **选择**：将预设应用到当前配置。

![预设预览](/img/betaflight_configurator_preset_2.png)

有关社区预设及其创建方法，参见[社区预设指南](/docs/wiki/guides/current/Community-Presets)。
