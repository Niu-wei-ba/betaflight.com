# 图传选项卡

在此配置图传发射器（VTX）的功率、频段、频道等参数。

:::note

使用 DJI 或 Walksnail 图传时可忽略此选项卡。本页仅适用于模拟 VTX 和 HDZero VTX。

:::

![VTX 选项卡](/img/betaflight_configurator_vtx_tab_zh-cn.png)

要使用此功能，必须在“端口”选项卡为 VTX 启用遥测协议（SmartAudio 或 Tramp）。

## 所选模式

| 选项                         | 说明                                                                                                                                                                                                     |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **直接输入频率**              | 启用后，地面站直接向 VTX 发送频率值，而不是通常的频段/频道。VTX 必须支持该功能。                                                                                                                          |
| **频段**                     | 选择使用的频段。                                                                                                                                                                                         |
| **频道**                     | 选择使用的频道。                                                                                                                                                                                         |
| **功率**                     | 选择使用的功率等级。                                                                                                                                                                                     |
| **维修站模式（Pit Mode）**   | 启用后，VTX 以极低功率工作，方便在工作台通电而不干扰其他飞手；通常有效距离小于 5 m。注意：部分协议（如 SmartAudio）无法在上电后通过软件启用维修站模式。                                                   |
| **维修站模式频率**           | 选择维修站模式下使用的频率。                                                                                                                                                                             |
| **低功率上锁**               | 启用后，飞行器处于未解锁状态时 VTX 使用最低可用功率，失控保护发生时除外。**开启**：未解锁即最低功率；**关闭**：始终使用设定功率；**保持直至解锁**：首次解锁前保持最低功率，首次解锁后切换至设定功率并保持。 |

## 当前值

| 选项                   | 说明                                |
| ---------------------- | ----------------------------------- |
| **设备准备就绪**       | 指示飞控是否正与 VTX 通信。         |
| **图传类型**           | 与 VTX 通信所用的遥测类型。         |
| **频段**               | 当前设置的频段。                    |
| **频道**               | 当前设置的频道。                    |
| **频率**               | 当前设置的频率。                    |
| **功率**               | 当前设置的功率等级。                |
| **维修站模式**         | 指示 VTX 是否处于维修站模式。       |
| **维修站模式频率**     | VTX 在维修站模式下设置的频率。      |
| **低功率上锁**         | 指示 VTX 是否处于未解锁低功率模式。 |

## 图传表

| 选项                           | 说明                                                                                                                                                                                                                                                                                                                                                                  |
| ------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **频段数量**                   | VTX 支持的频段数量。                                                                                                                                                                                                                                                                                                                                                  |
| **频段内的频道数**             | VTX 支持的每个频段的频道数量。                                                                                                                                                                                                                                                                                                                                        |
| **名称、字母、频率**           | 定义飞行器 VTX 可使用的所有频率。每个频段都需配置：**名称**，例如 `BOSCAM_A`、`FATSHARK`、`RACEBAND`；**字母**，引用该频段的简称；**工厂模式**，启用时 Betaflight 向 VTX 发送频段和频道号，VTX 使用内置频率表，此处频率仅用于 OSD 等显示；未启用时，Betaflight 直接发送此处定义的实际频率；**频率**，该频段的频率列表。请注意，并非所有频率在所有地区都合法。 |
| **功率级别数量**               | VTX 支持的功率等级数量。                                                                                                                                                                                                                                                                                                                                              |
| **功率级别**                   | 定义 VTX 可用功率。**值**是硬件制造商定义的功率等级值，可咨询制造商或参阅 [Betaflight 图传文档](/docs/wiki/guides/current/VTX#vtx-tables)示例；**标签**是显示标签，可用数值（`25`、`200`、`600`、`1.2`）、文字（`OFF`、`MIN`、`MAX`）或其组合。                                                                                                                 |

## 底栏

| 选项                    | 说明                                                                                                                                               |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| **保存**                | 将更改保存至飞控。                                                                                                                                 |
| **保存到文件**          | 将 VTX 设置保存为 JSON 文件。                                                                                                                      |
| **从文件加载**          | 从 JSON 文件加载 VTX 设置。                                                                                                                        |
| **从剪贴板加载**        | 从剪贴板加载 VTX 设置，内容必须是有效 JSON。                                                                                                       |
| **保存 Lua 脚本**       | 保存含 VTX 表配置的 `mcuid.lua`，可供 Betaflight TX Lua 脚本使用。1.6.0 及以后版本可直接使用；较旧脚本版本需将文件改名为与发射机中的模型名称一致。 |

更多 VTX 表和 CLI 命令信息，见 [VTX 文档](/docs/wiki/guides/current/VTX)。
