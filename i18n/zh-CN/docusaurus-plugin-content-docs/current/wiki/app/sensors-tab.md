# 传感器选项卡

传感器选项卡配置传感器硬件、加速度计和磁力计校准，并提供实时传感器数据图。它整合了之前分散在“配置”和“设置”选项卡中的传感器设置。

![传感器选项卡](/img/betaflight_configurator_sensors_tab_zh-cn.png)

## 传感器硬件

传感器硬件部分显示检测到的传感器并允许您启用或禁用它们。当检测到传感器时，其硬件名称会显示在切换开关旁边（例如“陀螺仪 1 - BMI270”、“加速度计 - BMI270”、“磁力计 - QMC5883”）。

| 设置         | 描述                                                                                                                |
| ------------ | ------------------------------------------------------------------------------------------------------------------- |
| **陀螺仪**   | 始终启用。显示检测到的陀螺仪硬件。在具有双陀螺仪的板上，“陀螺仪/加速度计”选择器可让您选择第一个、第二个或同时启用。 |
| **加速度计** | 启用或禁用加速度计。角度/水平模式、GPS 救机和磁力计航向校正所需。                                                   |
| **磁力计**   | 启用或禁用磁力计。启用后，其他对齐和校准选项将显示在下方。                                                          |
| **气压计**   | 启用或禁用气压计。用于高度估计。                                                                                    |
| **测距仪**   | 启用或禁用测距仪传感器（例如 MTF02P）。仅当固件包含测距仪支持时才可见。                                             |
| **光流**     | 启用或禁用光流传感器。仅当固件包含光流支持时才可见。                                                                |

### 飞控板朝向

补偿未平坦或方形安装到框架的飞控。输入滚动、俯仰和偏航校正角度（CLI 中为十分之一）。这些修正适用于陀螺仪、加速度计以及共享 FC 坐标系的任何传感器。

#### 飞控方向设置向导

在 API >= 1.48 的固件上，可以使用新的**飞控方向设置向导**来帮助设置正确的飞控板朝向。
当飞控按指示倾斜时，它通过分析加速度计和陀螺仪读数，提供逐步指导，以交互方式确定正确的横滚角、俯仰角和偏航角。

### 传感器朝向（旧版）

在 API < 1.47 的固件上，此处显示第一个陀螺仪、第二个陀螺仪和加速度计的单独朝向下拉列表。在较新的固件上，传感器朝向由系统自动处理，或通过“飞控板朝向”字段处理。

## 加速度计

### 校准

单击**校准**开始加速度计校准。将飞行器放置在平坦、水平的表面上，并且在校准期间不要移动它。正在进行时按钮显示“正在校准…”。如果加速度计尚未校准，则会出现警告横幅。

### 加速度计微调

使用横滚和俯仰微调值微调加速度计读数。这些小的修正可以弥补校准后留下的微小安装缺陷。

## 磁力计

当检测到并启用磁力计时，会出现磁力计部分。它提供对准配置、磁性参考检测、校准偏移编辑和校准工具。

有关完整的磁力计设置说明，请参阅[磁力计指南](/docs/wiki/guides/current/Magnetometer)。

### 磁力计对准

用于选择磁力计相对于机架的安装方向的下拉菜单。选项包括“默认”、CW90、CW180、CW270 及其用于倒置安装的翻转变体（常见于 GPS 集成磁力计）。选择“自定义”可显示用于任意方向校正的横滚、俯仰和偏航角度输入。

当您缓慢旋转飞行器时，**检测**按钮（API >= 1.47）通过分析传感器数据自动确定磁力计朝向。它收集样本，检测最可能的方向，并显示置信度。然后您可以应用或放弃结果。

### 磁偏角、磁倾角和磁场强度

这些字段显示用于航向校正的本地磁参考值。

| 字段       | 描述                                                                                                                               |
| ---------- | ---------------------------------------------------------------------------------------------------------------------------------- |
| **磁偏角** | 您所在位置的磁北和真北之间的角度差。可编辑 - 输入您的本地值，或单击 **检测** 以根据您的 GPS 或浏览器位置从在线地磁模型自动获取它。 |
| **倾角**   | 地球磁场浸入（或浸出）地面的角度。只读，由检测函数填充。                                                                           |
| **场强**   | 局部磁场的总强度，以纳特斯拉 (nT) 为单位。只读，由检测函数填充。                                                                   |

当自动检测到磁偏角时，会显示信息横幅。如果磁偏角值与检测到的值显着不同，则会出现警告横幅。

### 校准值

三个可编辑字段显示当前磁力计校准偏移（X、Y、Z）。从原始传感器读数中减去这些值，使数据以零为中心。

您可以手动编辑这些值，然后单击**保存数值**以通过 CLI 将它们写入飞控。这与全局**保存并重启**按钮是分开的，因为磁力计校准值是通过 CLI 命令存储的，而不是正常的 MSP 参数路径。

### 检查模式

单击**检查**按钮（眼睛图标）进入检查模式。这将打开 3D 球体可视化，显示以点绘制的实时磁力计读数。检查模式使用当前校准值并且不会修改它们。用它来：

- 验证当前校准是否准确（点应形成以原点为中心的球体）
- 确认磁力计方向正确
- 观察附近磁场干扰的影响

### 校准

单击**校准**开始磁力计校准。按钮旁边的下拉箭头提供不同的校准模式：

| 模式               | 描述                                                                                                                                                                                                    |
| ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **客户端校准（引导）** | 推荐。地面站会引导您完成 6 个方向（平放、倒置、机头向上、机头向下、左侧向上、右侧向上），并每 10 秒自动提示一次。显示 60 秒倒计时器。地面站使用球体拟合计算校准偏移。单击**保存校准**接受结果。 |
| **客户端校准（自由）** | 向各个方向自由旋转飞行器。地面站收集样本并计算偏移量。没有方向提示，您可以决定何时覆盖范围足够。                                                                                                        |
| **固件校准（旧版）**   | 触发固件的内置 30 秒校准。固件计算并自动保存偏移量。需要物理敲击或摇动，才能在 15 秒窗口内启动数据收集。                                                                                                  |

校准期间，3D 球体视图显示：

- **白点**随着您旋转飞行器而累积，形成一个球体
- 飞行器参考系中的**轴线**（红色 X、绿色 Y、蓝色 Z），显示每个轴的实时场强
- **白色机头方向线**从飞行器指向当前样本位置
- **橙色磁场参考箭头**显示局部磁场倾角
- **天球上的罗盘标签**（N、S、E、W）
- **绿点**显示当前固件校准偏移位置
- **灰点**显示计算出的球体中心（新的校准偏移）
- **质量得分**和覆盖率评估

质量指示器根据球体拟合残差和角度覆盖均匀性显示“良好”、“一般”或“差”。目标是实现完整的角度覆盖，点球内没有可见的间隙。

在校准过程中您可以：

- **取消**丢弃所有数据并返回空闲状态
- **清除**重置收集的样本并重新开始而不离开校准模式
- **保存校准**（引导/自由模式）将计算出的偏移写入飞控

### 旧版固件（API < 1.47）

在不支持 CLI-over-MSP 支持的旧固件上，仅显示一个 **校准** 按钮。这会触发固件的内置磁力计校准（相当于固件校准传统模式）。

## 实时传感器数据

单击“**显示实时传感器数据**”以显示传感器读数的实时图表。图表会自动缩放并显示来自所有启用的传感器的数据。可用的数据通道取决于哪些传感器处于活动状态以及所选的调试模式。

此处还显示了固件的调试值。在 Blackbox/Logging 选项卡中设置调试模式以选择公开哪些内部值。有关可用调试模式的参考，请参阅[调试模式指南](/docs/wiki/guides/current/Debug-Modes)。

## 保存并重启

单击页面底部的**保存并重启**以保留所有传感器配置更改（硬件启用、传感器朝向、飞控板朝向、加速度计微调）。请注意，磁力计校准偏移会通过“校准数值”部分中的**保存数值**按钮单独保存。
