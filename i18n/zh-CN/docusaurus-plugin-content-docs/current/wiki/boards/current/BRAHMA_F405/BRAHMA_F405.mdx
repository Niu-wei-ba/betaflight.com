## 输入/输出

- USB 接口：USB Type-C
- 电机输出：8 路
- UART：4 个
- I2C：有
- SWD：有
- SPI：有
- 3.3 V 输出：有
- 4.5 V（VBUS）输出：无
- 5 V 输出：2 A
- 9 V 输出：2 A
- 电流传感器：无
- 模拟 RSSI 输入：有
- LED 灯带输出：有
- 蜂鸣器输出：有

## 焊盘

### UART

| 名称   | 标签    | 备注     |
| ------ | ------- | -------- |
| UART 1 | TX1/RX1 | SBUS     |
| UART 2 | RX2     |          |
| UART 3 | RX3     | ESC 遥测 |
| UART 4 | TX4/RX4 |          |

### 电源

| 名称     | 标签 | 数量 | 备注           |
| -------- | ---- | ---- | -------------- |
| 3.3 V    |      | 1 个 | SWD 引脚定义中 |
| 5 V      | 5V   | 5 个 |                |
| 9 V      | 9V   | 1 个 |                |
| 电池电压 | VBAT | 1 个 |                |
