# SPRacingF7DUAL

SPRacingF7DUAL 是首款支持**同步双陀螺仪**和 **PID Audio** 功能的 Betaflight/Cleanflight 飞控。

它配备 F7（STM32F722）CPU、两颗支持 32 kHz 的陀螺仪、音频输出、FC/PDB、OSD、SD 卡、应答器、LED 灯带、8 路电机输出、5 个串口。F7 可在内部完成反相，因此无需外置反相器。此外还提供相机 OSD 控制、电流传感器、4 合 1 ESC 焊盘、BMP280 气压计、5 V BEC、PWM/模拟 RSSI、侧按式 Boot 按钮；如有需要，可使用随附胶套进行软安装。

这是一块功能非常完整的 FC。
