# CJMCU

CJMCU 是一块小型（80 mm）STM32F103 板卡，集成三轴磁力计（HMC5883L）和加速度计/陀螺仪（MPU6050）。

本板没有板载 USB 转串口芯片，需要使用外置适配器。

# 硬件修订版

| 修订版 | 说明                                                            |
| ------ | --------------------------------------------------------------- |
| 1      | LED1 旁没有 BOOT 跳线焊盘，使用蓝色和红色 LED。                 |
| 2      | LED1 旁的 BOOT 跳线焊盘已预焊排针和跳线帽，使用绿色和红色 LED。 |

版本 2 板卡从固件 v1.4.0 开始才受支持。**切勿**向版本 2 板卡刷写更早版本的固件。

# 引脚

## 接收机连接

| 引脚标签 | 说明                  |
| -------- | --------------------- |
| PA0      | RC 通道 1             |
| PA1      | RC 通道 2             |
| PA2      | RC 通道 3 / USART2 TX |
| PA3      | RC 通道 4 / USART2 RX |
| VCC      | 电源（见说明）        |
| GND      | 地                    |

注意：VCC RX 引脚未稳压，会输出板卡接收到的输入电压。例如，若使用 5 V 串行连接，它会输出 5 V。使用对电压敏感的接收机时请务必注意。第 1 列最上方、RX GND 引脚正下方可取得稳压 3.3 V 电源。

## 串行连接

USART1（以及电源）位于以下引脚：

| 引脚标签 | 说明        |
| -------- | ----------- |
| TX1      | UART1 TX    |
| RX1      | UART2 RX    |
| GND      | 地          |
| 3V3      | +3.3 V 电源 |
| 5V       | +5 V 电源   |

USART2 位于以下引脚：

| 引脚标签 | 说明      |
| -------- | --------- |
| PA2      | USART2 TX |
| PA3      | USART2 RX |

## 电源连接

| 引脚标签 | 说明                  |
| -------- | --------------------- |
| Power +  | 电源，最大 1 节 3.7 V |
| Power -  | 地                    |

## 电机连接

在标准 QUADX 配置中，电机映射如下：

| Cleanflight | CJMCU  |
| ----------- | ------ |
| 电机 1      | 电机 3 |
| 电机 2      | 电机 2 |
| 电机 3      | 电机 4 |
| 电机 4      | 电机 1 |

因此，最简单的接线方式为：

- 电机 1 -> 顺时针
- 电机 2 -> 逆时针
- 电机 3 -> 顺时针
- 电机 4 -> 逆时针

若使用 Hubsan x4/Ladybird 电机，顺时针电机的导线为蓝色（GND）/红色（VCC），逆时针电机的导线为黑色（GND）/白色（VCC）。也就是说，每个电机都有一根不遵循标准红/黑 VCC/GND 配色的导线，可用于识别极性。

按上述方式接线后，板上的 Motor1/Motor2 位于机头前方。

# 连接 USB 转串口适配器

需要一个 USB -> 串行 UART 适配器，连接方式如下：

| 适配器       | CJMCU                    |
| ------------ | ------------------------ |
| 3.3 V 或 5 V | 对应的 3.3 V 或 5 V 引脚 |
| RX           | TX                       |
| TX           | RX                       |

首次连接时，板卡应上电并进入引导加载程序模式。若未发生，请检查充电开关是否置于 POW。刷写完成后，可通过 Cleanflight Configurator 访问板卡、修改设置或刷写新固件。

警告：如果已连接电机且板卡进入引导加载程序，电机约 20 秒后会开始转动。建议在刷写完成前不要连接电机。

# 刷写

刷写板卡的步骤：

- 打开 Cleanflight Configurator。
- 从列表中选择最新的 CJMCU 固件。
- 选择“从网络加载固件”，等待固件下载完成。
- 勾选“无重启序列”和“全芯片擦除”。
- 将 USB -> 串行适配器连接至板卡。
- 在左上方列表中选择 USB-UART 适配器。
- 单击“烧写固件”。
- 日志框中应显示“Programming: SUCCESSFUL”。
- 单击“Connect” -> 应打开“Initial Setup”选项卡，并显示四轴飞行器的传感器数据。
- 断开四轴飞行器电源，并短接两个“BOOT0”引脚。修订版 1 需要焊接短接；修订版 2 可将随附跳线帽插到两个预焊引脚上。这样可避免下次启动进入引导加载程序；若发生问题，只需断开这两个引脚即可再次进入引导加载程序并重新刷写。无法覆盖引导加载程序本身。

# 充电

CJMCU 板载 TP4056 锂电池充电 IC；通过连接至 5 V 串行引脚的 5 V 电源，可为 1S 电池以 1 A 充电。

为已连接的电池充电：

- 将电源开关设为 OFF。
- 将充电开关设为 CHG。
- 将 1S 电池接到电池引脚。
- 将 5 V 电源接到 5 V 串行引脚。

当电池达到 4.2 V，或电池电压高于充电器输入电压时，充电将结束。

附近两个 LED 显示充电状态：

| 状态       | 绿灯 | 红灯 |
| ---------- | ---- | ---- |
| 充电中     | 亮   | 灭   |
| 已完成     | 灭   | 亮   |
| 未连接 5 V | 灭   | 灭   |
| 未连接电池 | 闪烁 | 亮   |

# 实用提示

- 若仅使用 4 通道接收机，可在辅助配置选项卡中为某个 AUX 通道添加约 1500 的“Horizon”模式范围，使其始终开启。
- 启用 `feature MOTOR_STOP` 有助于在坠机后停止电机，避免机体倒扣时继续旋转。
- 电量较低时，四轴飞行器可能开始轻微跳动。若飞行表现异常，请检查电池电量。
