# Seriously Pro SP Racing F3 MINI

Seriously Pro Racing F3 Mini (SPRacingF3Mini) 是第二款专为 Cleanflight 设计的飞控。它是唯一支持所有最新 Cleanflight 功能的 FC，也是唯一具有完整功能集的迷你板，也可以安装在迷你框架以及标准 36x36mm 安装孔中。凭借新的应答器功能，它是唯一真正适合比赛的板。

其完整的功能集、尺寸和重量使其非常适合集成到 250 克以下的无人机中，以符合美国无人机法规。

直接从 SeriouslyPro / SP Racing 或官方零售商购买板卡，有助于资助 Cleanflight 开发；官方零售商始终列于 SeriouslyPro.com。

完整详细信息可在网站上找到：

http://seriouslypro.com/spracingf3mini

## 硬件特性

- 下一代 STM32 F3 处理器，具有硬件浮点单元，可实现高效的飞行计算和更快的 ARM-Cortex M4 内核。
- 用于 Blackbox 飞行日志记录器的 MicroSD 卡插槽 - 优化您的调整并查看设置结果，无需猜测。
- 内置竞赛应答器，可记录比赛圈速。
- 板载调节器 (BEC)，用于为 FC、接收机和小型伺服系统供电。
- 采用最新的加速计、陀螺仪、磁力/指南针以及气压/高度传感器技术。
- 2 个按钮，用于绑定 Spektrum 卫星接收机、激活 USB 引导加载程序模式或重置配置。更多未来用途即将到来！
- 使用排针连接所有主要连接，以实现出色的耐碰撞性。使用直角或直排针。
- 完整 I/O：可同时连接 USB、OSD、SmartPort、SBus、GPS、LED 灯带、电池监测、声纳和 8 路电机。
- 8 条用于 ESC 和舵机的 PWM 输出线。安排方便在标准排针上接线。
- 支持直接连接 SBus、SumH、SumD、Spektrum1024/2048、XBus 接收机。无需外部逆变器（内置）。
- 支持通过 3 针通孔 JST-ZH 连接器直接连接 3.3v Spektrum 卫星接收机。
- 支持直接连接 1-5 通道并行 PWM 接收机\*1。
- 专用 PPM 接收机输入。
- 3 个串行端口 - 不与 USB 插座共享。
- 遥测端口（通过排针或 USART2 JST-SH 插座）。
- 微型 USB 插座。
- 可编程 LED 的专用输出 - 非常适合定向、赛车和夜间飞行。 （目前与应答器互斥）。
- 专用 I2C 端口，用于连接 OLED 显示屏，无需飞行电池。
- 电池电压和电流监控。
- RSSI 监控（模拟或 PWM）。
- 蜂鸣器端口用于发出声音警告和通知。
- 开发人员友好的调试端口（SWD）和启动模式选择，不可破解的启动加载程序。
- 对称设计，布线超级整洁。
- JST-SH 插座仅适用于 I2C/UART2 和 SWD。
- 气压计安装在板的底部，方便隔风。
- 通过 USB 或串行端口刷新。
- 可堆叠设计 - 非常适合与 OSD 和配电板集成。
- 模块化设计 - 板的核心尺寸为 36x22mm，但能够使用标准 30.5mm 安装孔进行安装。
- 3v、5v 和状态 LED 指示灯，便于诊断。
- 蚀刻的 Cleanflight 和 #RB 标志。

\*1 - PWM 接收机必须使用 3.3v 输出。仅适用于多旋翼模式。使用电机输出 5-8 和 PPM 引脚作为 RC 1-5 输入。

#### 主要部分

主要部分是板的方形部分，带有 30.5mm 安装孔。

### 左侧 IO（从前到后）

| 引脚 | 功能                         | 备注 |
| ---- | ---------------------------- | ---- |
| 1    | RX3                          | 方垫 |
| 2    | TX3                          | 圆垫 |
| 3    | PWM8 / SoftSerial 1 RX / RC4 | 方垫 |
| 4    | PWM7 / SoftSerial 1 TX / RC3 | 方垫 |

PWM7 和 PWM8 的左侧还有 2 个引脚 - 从左到右：GND、VIN、PWM7/8。RX3 右侧还有两个通孔。使用 RX3 和 2 个孔连接 Spektrum Satellite 3v 接收机的 JST-ZH 连接器。

### 右侧 IO（从前到后）

| 针  | 功能       | 笔记         |
| --- | ---------- | ------------ |
| 1   | RSSI       | 圆形焊盘/PWM |
| 2   | CURRENT    | 圆形焊盘     |
| 3   | PWM6 / RC2 | 方形焊盘     |
| 4   | PWM5 / RC1 | 方垫         |
| 5   | T1         | 圆垫         |
| 6   | R1         | 圆垫         |
| 7   | 5v         | 圆垫         |
| 8   | GND        | 圆形焊盘     |
| 9   | 5V         | 圆形焊盘     |

在 PWM5 和 PWM6 的右侧还有 2 个引脚 - 从左到右：PWM5/6、VIN、GND。

引脚 8/9/10 允许为 PPM 接收机连接标准 3 引脚电缆。
引脚 8/7/6 允许为 SBus/SerialRX 接收机 (5v) 连接标准 3 引脚电缆。

### 应答器部分

该板的应答器部分最多可连接两个随附 IR LED，用于比赛计时应答器系统。可保持连接、堆叠在主板上方或下方，或拆下后使用线缆连接。

应答器部分有一个小的使能跳线，在使用红外应答器功能之前必须用焊料桥接该跳线。

#### 左下和右下应答器 IO（从左到右）

用于 LED/IR。应答器板安装孔处排列成正方形的 4 个引脚在板的左右两侧具有相同的引脚排列。它们是相同的，因此在重新定位应答器部分时，可以在任一侧连接 4 针电缆。

A - 底行（朝向板边缘）
B - 顶排（朝向棋盘中心）

| 针  | 行   | 功能         | 笔记 |
| --- | ---- | ------------ | ---- |
| 1   | 一个 | 常闭         | 方垫 |
| 2   | 一个 | 接地         | 圆垫 |
| 3   | 乙   | 5v           | 圆垫 |
| 4   | 乙   | LED_STRIP/IR | 圆垫 |

### 左下和右下红外

一些表面贴装组件的应答器部分的左右两侧都有两个销孔。它们用于连接最多两个红外 LED。
A - 底行（朝向板边缘）
B - 顶排（朝向棋盘中心）

| 针  | 行   | 功能  | 笔记 |
| --- | ---- | ----- | ---- |
| 1   | 一个 | 红外+ | 圆垫 |
| 2   | 乙   | 红外- | 方垫 |

注意：早期板上的丝印有 IR+ 和 IR- 互换。无论丝印如何，都按上述连接。

### 开关部分

开关部分的底部有 4 个垫子。开关部分可以重新定位并通过 4 路电缆重新连接到主部分。

| 开关部分 | Pad On 主要部分垫 |
| -------- | ----------------- |
| 一个     | 回声              |
| 乙       | 触发              |
| 接地     | 任何接地          |
| 3v3      | 任何 3.3v         |
