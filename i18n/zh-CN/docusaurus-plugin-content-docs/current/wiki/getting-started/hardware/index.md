# 硬件

飞控（FC）要实现完整功能，需要传感器、电源管理和各类输入输出组件协同工作。本页介绍 FC 的主要组成部分、各自的作用，以及常见的器件选型。

## 核心组件与通用 I/O

### MCU

MCU（微控制器单元）是 FC 的主芯片，可视为飞控的“大脑”。它运行飞控软件、处理所有传感器数据，并通过大量 ~~魔法~~ 运算生成发送给电调（ESC）的控制信号，从而驱动电机。

大多数 FC 使用基于 ARM Cortex 架构的 MCU，最常见的是 STM 的产品；近年来也有部分 FC 采用 ArteryTek 的 MCU。

:::info

你可能见过 `F4`、`F7`，或更具体的 `F411`、`F405`、`F722` 等名称。这些是不同 MCU 产品线的型号标识，其 Flash 容量、RAM 和时钟频率各不相同。通常型号数字越高，MCU 性能越强。

以下按性能由低到高排列：

- **STM32F411**：价格低、体积小，常见于入门级和一体式（AIO）FC；但性能弱于大多数其他 MCU。
- **STM32G473**：STM 较新的选择，性能略高于 F411，但生态和成熟度较低。
- **STM32F405**：目前最常见的选择之一，在性能与成本之间取得了良好平衡，已长期广泛使用。
- **AT32F435**：ArteryTek 面向 F405 的新替代方案，引脚直接兼容，性能略高。
- **STM32F722**：很受欢迎的高端选择，但 Flash 容量有限。如今可借助 Cloud Build 节省大量空间，因此通常不是大问题。
- **STM32F745**：性能略高于 F722，Flash 空间也更充足，适合高端配置且无需过度堆料的构建。
- **STM32H743**：性能最强、价格也最高。大多数装机并不需要它，但适合追求极限规格的场景。

:::

### 稳压器

稳压器负责将输入电压升压或降压至 MCU 和其他组件所需的电压。MCU 与板载传感器通常工作在 3.3 V；接收机、相机和部分低功率 VTX 等外设通常使用 5 V。若要驱动大功率 VTX 或大多数数字图传设备，FC 上可能还会提供 9 V、10 V 或 12 V 稳压器，其中如今更推荐 12 V。

:::caution

一个或多个稳压器过载时，其他电子设备可能工作异常、完全无法工作，甚至损坏稳压电路；某些情况下，连接到该稳压器的组件也会受损。因此，必须确认稳压器的额定电流能够满足所用组件的总电流需求。大多数常规硬件配置不会出现问题，但例如连接大量 LED 时，5 V 稳压器就可能过载。

:::

### OSD

要在模拟视频中显示 OSD，FC 会直接接收来自相机的原始视频信号，并将其送入 MAX7456 芯片处理。AT7456E 是可直接替代的兼容芯片。这类芯片会将字体文件中定义的基础字符叠加到画面上，再把处理后的模拟视频信号输出给 VTX 发射。

:::note

数字图传系统不需要模拟 OSD 芯片。OSD 信息会发送到数字 VTX，并与视频一起传输，最终只在护目镜中叠加显示。因此可以获得干净的视频录像，也能导出 OSD 数据等。

:::

## 姿态与定位传感器

### 陀螺仪与加速度计

陀螺仪和加速度计用于测量 FC 的姿态与角速度。陀螺仪测量角速度，加速度计测量姿态。

- 陀螺仪用于不需要 FC 知道自身空间姿态、只需角速度的飞行模式，例如 Acro 模式。
- 加速度计用于需要获知空间姿态的飞行模式或功能，例如 Angle/Horizon 模式和 GPS 救机。

:::info

几乎所有 FC 都使用单颗封装的陀螺仪/加速度计组合芯片，通常通过 SPI 通信。与 MCU 一样，可选型号很多：

- **Invensense MPU6000**：已使用多年，至今仍被许多人视为最佳选择。它曾一度停产，推动了其他替代方案的发展，现在已恢复供应。
- **Bosch BMI270**：MPU6000 的优秀替代品。为取得最佳表现，它的运行频率低于多数其他芯片，例如 MPU6000 为 8 kHz，而 BMI270 为 3.2 kHz。
- **Invensense ICM42xxx/ICM20xxx**：MPU6000 短暂停产期间逐渐普及的新选择。早期表现较弱，但在 4.4.1 之后已达到甚至超过 MPU6000 的水平，无需因飞控采用此类芯片而担心。

:::

### 气压计

气压计测量空气压力，可据此计算传感器所在的高度。它用于 GPS 救机模式中的安全着陆等功能，未来还计划用于定高及更高级的功能。

:::info

大多数 FC 最常见的气压计是 Bosch BMP280。BMP180、MS5611 和 DPS310 等型号也会出现，但并不常见。

:::

:::caution

气压计芯片封装本身必须带有开孔，才能读取空气压力，见下图。可在上方放置一小块开孔泡棉，以获得更稳定的测量结果。使用三防漆时，务必不要覆盖该开孔，否则气压计将无法工作。

:::

### 磁力计

磁力计测量周围磁场，本质上就是电子罗盘。它用于判断 FC 相对于地球磁场的朝向。在当前的 4.4.1 版本中，磁力计尚未完全用于辅助高级 GPS 救机功能，但后续版本已有相关计划。

:::caution

正如名称所示，磁力计对磁场非常敏感，包括来自电机、ESC、VTX 和接收机的电子干扰。使用外置磁力计时，应尽可能让它远离这些干扰源，GPS 模块通常会集成磁力计；同时应仔细校准，才能获得可用结果。

:::

### GPS

GPS 借助环绕地球运行的全球卫星定位系统获取 FC 的位置和运动速度，也可结合 GLONASS、Galileo 等其他卫星系统。它能为 FC 提供准确的位置和速度数据，可显示在 OSD 上、传回遥控器以便坠机后定位，或通过 GPS 救机让飞行器返回。

:::danger

市面上的大多数 GPS 模块基于 u-blox M8 芯片，但也有采用 M10 芯片的新模块。M10 的性能明显优于 M8，但有一个前提：在 4.5.0 之前，Betaflight 不支持自动配置 M10，必须使用 u-center 软件手动配置；配置不当会导致性能不可靠。从 4.5.0 开始，无需再手动配置。

:::
